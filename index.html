LibNLI

Natural Language Interface for Library API Usage

Try LibNLI for Apache POI

Overview

LibNLI navigates the learning process of Java libraries. Developers browse domain concepts and programming tasks in natural language for a given library. Each concept or task is mapped to its implementation with illustrative examples.

Dataset

To construct current version of LibNLI for Apache POI, we analyzed source code , related StackOverflow posts and 623 client repositories. We apply the novel code clone detector SourcererCC tool to cluster client code and present API examples.
